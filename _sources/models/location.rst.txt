===============
Location Filter
===============

The location filter module is a utility module, that is usually accessed via 
The `Entry.neighbors <metacatalog.models.Entry.neighbors>`_ function or the 
`find_entry <metacatalog.api.find_entry>`_ API.
However, the uitl module is importable and the functions can be used directly 
for more advanced use cases.
